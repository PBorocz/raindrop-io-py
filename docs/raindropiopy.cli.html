raindropiopy.cli package

Top level CLI dunder init, primarily constants used across all commands.

raindropiopy.cli.goodbye(console) → None

Print a nice thank you message on successful exit from any command/event-loop.

raindropiopy.cli.make_italic(str_)

Use Rich’s rich text to make string italic.

raindropiopy.cli.options_as_help(options: list[str], depth: int = 1) → str

Return the list of options in a nice format for display.

Input options might be: [“Action” , “SomethingElse” , “Done”] first, we convert to [“(A)ction”, “(S)omethingElse”, “(D)one”] then we render to “[italic](A)nAction[/italic], “

Similarly, when called with: [“title”, “tag”, “back”] AND depth of 2: we convert to [“(ti)tle”, “(ta)g”, “(b)ack”] and then render to “[italic](ti)tle[/italic] [italic](ta)g[/italic]…”

raindropiopy.cli.prompt(sub_levels: tuple[str] | None = ()) → str

Render a command prompt to any number of ‘levels’.

With no sub-levels, prompt is: “> ” With sub-levels (“A”, “SubA”): “A> SubA> “
